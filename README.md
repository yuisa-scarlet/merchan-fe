## Instalasi


Persyaratan [Node.js](https://nodejs.org/) versi LTS.

Klon repositori ini.

```
git clone https://github.com/yuisa-scarlet/merchan-fe.git
```

Masuk ke direktori.

```
cd merchan-fe
```

Unduh pustaka yang diperlukan.

```
npm install --legacy-peer-deps
```

## Running project

Jalankan project

```
npm run dev
```
